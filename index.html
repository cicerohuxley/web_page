O que são Clients?

Resposta:

Conhecido também como Lado do cliente ou front-end, é o lado do site onde há renderização das paginas web onde usuários poderam ver e até mesmo interagir com as páginas

Resposta:

É onde o conteúdo da internet está fisicamente, em outras palavras, para que eu consiga obter uma resposta de uma pagina web, ela tem que existir em algum lugar, esse é o lado do servidor, o computador do lado do cliente apenas faz uma cópia desse conteúdo para que o usuário possa usa-lo.

Como é uma estrutura básica de um HTML?

Resposta:

A estrutura básica de uma página html é composta por três partes, a tag html que envolve todo o documento da página em questão, a tag head onde é adicionados configuraçãoes diversar para a página, está ultima não aparece para o client quando o arquivo é rodado em um servidor, e por ultimo o body onde o próprio nome já diz, é o corpo de nossa página onde será renderizado para o cliente

Entendendo a diferença entre listas ordenadas e não ordenadas

Resposta:

A diferença entre listas ordenadas e não ordenadas está clara nos nomes, pois listas ordenadas, referece-se que a ordem é de suma importância. Já as não ordenadas é o oposto, não há importância na ordenação dos items contidos na lista
